# Ubuntu 20.04

## Git

### Installation

```sh
$ sudo add-apt-repository ppa:git-core/ppa

$ sudo apt update

$ sudo apt install git
```

### Setting

```sh
$ git config --global user.email "18248972+junlico@users.noreply.github.com"

$ git config --global user.name "junlico"
```
